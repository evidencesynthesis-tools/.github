## Evidence Synthesis Tools

A curated directory of **140 strictly open-source tools (up to 2026)** covering the complete evidence synthesis workflow.

### Scope

Includes tools for:

* Searching and citation retrieval
* Screening and study selection
* Reference management and deduplication
* Data extraction and PDF processing
* Risk of Bias assessment
* Meta-analysis and statistical synthesis
* Evidence mapping, automation, and AI-assisted workflows

### Inclusion Criteria

Tools must:

* Have an **open-source license**
* Provide a **public code repository**
* Be **non-proprietary**
* Be relevant to systematic reviews or evidence synthesis

### Why Open Source?

Supports **transparency, reproducibility, reuse, and global access** for research and open science.

### Contribute

Submit tools via GitHub Issues, Discussions, Mail or Pull Requests.
